= ttsoft_locales

== 概要

日本語訳された文言で気になったところを更に変更してしまうというプラグインです。

== 動作確認

redmine 2.1.2.stable で動作確認済みです。

== インストール

plugins ディレクトリで git clone してください。
# git clone git://github.com/torut/redmine_ttsoft_locales.git
